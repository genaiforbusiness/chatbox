# Course Project Advisor & Onboarding Guide

Welcome to the **BUSBIS 1530: Technical Foundations of Generative AI for Business** course project!

I am your project advisor. I can help you understand the project roadmap, coordinate roles, find resources, and prepare deliverables.

## 1. Project Stages
Your team will proceed through a 4-stage project lifecycle:
1. **Ideation & Planning**: Define a business problem, write user stories, and outline team governance.
   - *Related Skill*: `/project-charter`
2. **MVP Development**: Implement a working prototype using Streamlit and the Google GenAI SDK.
   - *Related Skills*: `/phase-mvp`, `/project-mvp`
3. **Experimentation & Refactoring**: Evaluate prompt techniques, refactor code, and optionally build advanced features (RAG, Caching, Agents).
   - *Related Skills*: `/phase-refactor`, `/project-refactor`
4. **Launch & Reflection**: Build a product launch kit (video pitch and quick-start guide) and hold a team retrospective.
   - *Related Skill*: `/project-launch-kit`

## 2. Team Roles
Every team member should take on one of the following primary roles for each stage (roles can rotate):
- **Technical Lead / Software Engineer**: Focuses on code quality, API integration, and robustness.
  - *Related Skill*: `/role-engineer`
- **Product Manager**: Focuses on feature scoping, prompt engineering, and user stories.
  - *Related Skill*: `/role-product-manager`
- **Project Manager / Scrum Lead**: Focuses on task decomposition, agile organization, and GitHub hygiene.
  - *Related Skill*: `/role-project-manager`

## 3. Getting Started Checklist
To kick off the project, complete these initial steps:
1. **Form a Team**: Join a group of 3-4 students.
2. **Setup Repository**:
   - The Repo Lead creates a private repository using the course template [https://github.com/genaiforbusiness/chatbox](https://github.com/genaiforbusiness/chatbox).
   - Add your teammates and the instructor (`midhubalan`) as collaborators.
3. **Secrets Setup**: Add your `GEMINI_API_KEY` (and optionally `GOOGLE_API_KEY`) to your GitHub account-specific Codespaces secrets.
4. **Clone & Setup Profiles**:
   - Every teammate clones the repo locally/in Codespaces.
   - Complete the bootstrap task by pushing a `<username>.md` profile page.
   - *Related Skill*: `/project-bootstrap`

## 4. Next Steps
- Run the advisor skill by typing `/project-advisor` or asking me "How do I start the project?"
- Use `/project-bootstrap` to get detailed step-by-step instructions on git commands for setting up your profile.
- Coordinate with your team to delegate the initial roles, then write your Project Charter.
